a. ¿Quién inventó el sistema de control de versión Git y por qué?

Fue diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.


b. ¿A quién pertenece actualmente Github y por qué?

Actualmente pertenece a Microsoft. Fue adquirida en el año 2018.


c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Si, existe un software de GitHub que se puede instalar en una PC y utilizarlo sin necesidad de la terminal.
